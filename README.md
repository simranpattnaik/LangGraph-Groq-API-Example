🚀 LangGraph + Groq API Example

This project demonstrates how to build a simple AI workflow using LangGraph and Groq API. It showcases how graph-based orchestration can manage multi-step reasoning with Large Language Models (LLMs).

Instead of a traditional prompt → response approach, this project structures AI logic into a workflow of nodes and edges.
