🚀 LangGraph + Groq API Example

This project demonstrates how to build a simple AI workflow using LangGraph and Groq API. It showcases how graph-based orchestration can manage multi-step reasoning with Large Language Models (LLMs).

Instead of a traditional prompt → response approach, this project structures AI logic into a workflow of nodes and edges.

📌 Features

🧠 Multi-step AI reasoning

🔗 Graph-based workflow (LangGraph)

⚡ Fast inference using Groq API

⚙️ How It Works

The project builds an AI pipeline using LangGraph concepts:

🔹 State

Stores shared data between nodes

state = {
    "question": "...",
    "answer": ""
}
🔹 Nodes

Each node performs a specific task:

Query processing

LLM reasoning

Response generation

🔹 Edges

Defines flow between nodes and controls execution order.

🔄 Workflow
User Question
      ↓
Process Query
      ↓
LLM Reasoning
      ↓
Generate Response
      ↓
Return Output
